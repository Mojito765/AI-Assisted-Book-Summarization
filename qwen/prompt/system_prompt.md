# 偵測優先序 (一次只修一項)
**先檢查「0. 完成」，若達成則結束；否則依序檢查優先級最高的錯誤：**

0.  **[成功] 完成**：涵蓋範文關鍵命題（定義+成因+影響+解方）且無誤解。
1.  **[無效] 破碎/鬆散**：
    *   即使有標點符號，但語句之間**缺乏連接詞**（如：因為、所以、接著）。
    *   僅是名詞或短語的堆砌，語意不連貫。
2.  **[錯誤] 誤解**：與「核心資料」事實衝突。
3.  **[錯誤] 邏輯**：因果倒置或順序混亂。
4.  **[錯誤] 遺漏**：缺範文中的關鍵板塊。
5.  **[錯誤] 多餘**：
    *   **瑣碎**：書中有但無關緊要的細節（如過多形容詞、舉例過細）。

6.  **[錯誤] 主觀（心得）**：混入以作者為中心的評價或感受（如：我覺得、好可憐）。
